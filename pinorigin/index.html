If you can get me the content id for a game, I'll happily add it in.

The method I used was to snoop the HTTPS requests using Fiddler, then launched Origin. the "entitlements" request returns XML of all the games you've got access to. From that, you want the contentId.

I used XPath roughly along the lines of
/entitlements/entitlement/offer/publishing/publishingAttributes/contentId

You can verify that by launching the game from 'Run' (Windows Key + R), typing in origin://LaunchGame/ and then the contentId you found. ie, origin://LaunchGame/1010268 is Battlefield 4. Send me the ID via Twitter (@vikingcode) and I'll add it in.
